# SentenceTransformer based on sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2

This is a [sentence-transformers](https://www.SBERT.net) model finetuned from [sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2](https://huggingface.co/sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2). It maps sentences & paragraphs to a 384-dimensional dense vector space and can be used for semantic textual similarity, semantic search, paraphrase mining, text classification, clustering, and more.

## Model Details

### Model Description
- **Model Type:** Sentence Transformer
- **Base model:** [sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2](https://huggingface.co/sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2) <!-- at revision e8f8c211226b894fcb81acc59f3b34ba3efd5f42 -->
- **Maximum Sequence Length:** 256 tokens
- **Output Dimensionality:** 384 tokens
- **Similarity Function:** Cosine Similarity

## Usage

### Direct Usage (Sentence Transformers)

First install the Sentence Transformers library:

```bash
pip install -U sentence-transformers
```

Then you can load this model and run inference.
```python
from sentence_transformers import SentenceTransformer

# Download from the 🤗 Hub
model = SentenceTransformer("sentence_transformers_model_id")
# Run inference
sentences = [
    'Davacı, takip alacaklı olarak davayı açtı. Davacı şirkete haciz ihbarnamesi gönderildi ve itiraz edilmediği için takip alacaklı olarak kabul edildi. Davacı, dava dışı takip borçları ve herhangi bir borcu olmadığını iddia etti. Mahkeme, dava konusu şeyin yüzde yirmisi oranında tazminata mahkum etti.',
    'Davayı İK 89/3 ve 89/4 maddeleri uyarınca açıldı. Mahkeme, dava konusu şeyin yüzde yirmisinden aşağı olmamak üzere tazminata mahkum edildi. Mahkeme, dava konusu şeyin yüzde yirmisi oranında tazminata mahkum edildi.',
    'Mahkeme, davacı tarafından verilen imalat bedellerinin davalı tarafından ödenmediğini, davacı tarafından işin tamamlanmadığını ve davalı tarafından işin tamamlanması için gereken süre verilmediğini tespit etti. Davacı tarafından verilen imalat bedellerinin davalı tarafından ödenmediği ve davalı tarafından işin tamamlanmadığı için davacı tarafından davayı fesih talep edildi.',
]
embeddings = model.encode(sentences)
print(embeddings.shape)
# [3, 384]

# Get the similarity scores for the embeddings
similarities = model.similarity(embeddings, embeddings)
print(similarities.shape)
# [3, 3]
```
